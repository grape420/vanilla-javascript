호이스팅 : 
    스코프 내부에서 변수 선언은 최상위에 선언된 것처럼 사용한다.
    {} - 최상단으로 끌어 올려진 같은 현상

    변수 선언
    1. 선언 : 변수 등록 ( 변수는 스코프가 참조하는 대상 ) 
    2. 초기화 : 변수 공간의 메모리 할당 ( undefined )
    3. 할당 : 값을 할당

    var
    1. 선언 및 초기화
    2. 할당

    let
    1. 선언
    2. 초기화
    3. 할당

    const
    선언 + 초기화 + 할당

    var : 함수 스코프
    let, const : 함수스코프, 블록스코프 == {}
    function() {}
    if{} for{}...
