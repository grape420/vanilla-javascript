if : 만약에 ~ 이면 
    예) 만약에 수업끝나고 친구를 만나면 저녁을 먹는다.
    if (친구를 만나면) {
      저녁을 먹는다.
    }

    if (조건식) : 관계, 논리 - true / false + 산술
      결과가 한 줄이면 {} 생략 가능
    if (조건식) {
      true 결과
    } else {
      false 결과
    }

    예) 만약에 수업끝나고 친구를 만나면 저녁을 먹는다. 그렇지 않으면 집에 간다.
    if (친구를 만나면) {
      저녁을 먹는다.
    } else {
      집에 간다.
    }
