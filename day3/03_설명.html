if : 만약에 ~ 이면 
    예) 만약에 수업끝나고 친구를 만나면 저녁을 먹는다.
    if (친구를 만나면) {
      저녁을 먹는다.
    }

    if (조건식) : 관계, 논리 - true / false + 산술
      결과가 한 줄이면 {} 생략 가능
    if (조건식) {
      true 결과
    } else {
      false 결과
    }

    예) 만약에 수업끝나고 친구를 만나면 저녁을 먹는다. 그렇지 않으면 집에 간다.
    if (친구를 만나면) {
      저녁을 먹는다.
    } else {
      집에 간다.
    }

    다중 if문 
    철수를 만나면 커피를 마시고
    영희를 만나면 술을 마시고
    재석이를 만나면 고기를 먹고
    호동이를 만나면 저녁을 먹고
    아무도 안 만나면 집에 간다.
    if (철수를 만나면) {
      커피를 마신다.
    } else if (영희를 만나면) {
      술을 마신다.
    } else if (재석이를 만나면) {
      고기를 먹는다.
    } else if (호동이를 만나면) {
      저녁을 먹는다.
    } else {
      집에 간다.
    }
